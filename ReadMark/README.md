# ReadMark - 阅读指示器

一个轻量的 Chrome 扩展，帮助你在长文章阅读时追踪阅读位置，逐段高亮导航，告别"读到哪了"的困扰。

## 功能特性

- **段落高亮导航**：激活后自动识别页面有效段落，逐段高亮当前阅读位置
- **智能定位**：激活时自动定位到距离当前视口中心最近的段落，从你正在看的地方开始
- **多种导航方式**：键盘方向键 `↑` / `↓` 或右侧浮层按钮均可切换段落
- **SPA 兼容**：支持动态渲染的单页应用，段落列表在需要时自动重建
- **一键开关**：点击扩展图标切换激活/停用状态，不影响页面正常浏览

## 效果预览

激活后，当前段落会显示绿色半透明背景高亮，页面右侧出现 ↑↓ 浮层导航按钮。

## 安装方式

1. 克隆或下载本仓库到本地
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目目录
5. 扩展安装完成，工具栏会出现扩展图标

## 使用方法

1. 打开任意包含长文章的网页
2. 点击工具栏中的扩展图标，激活阅读模式
3. 使用键盘 `↑` / `↓` 或右侧浮层按钮在段落间切换
4. 再次点击扩展图标退出阅读模式

## 项目结构

```
ReadMark/
├── manifest.json   # 扩展配置（Manifest V3）
├── background.js   # Service Worker，处理图标点击事件
├── content.js      # 核心逻辑，段落收集、高亮、键盘导航
├── content.css     # 高亮样式与浮层按钮样式
└── icon.png        # 扩展图标
```

## 技术实现

- **Manifest V3**：采用最新 Chrome 扩展规范，Service Worker 后台脚本
- **段落识别策略**：优先识别 `<p>` 标签，不足时 fallback 到 `article/main` 下的 `div`，过滤导航栏、页眉页脚等无效区域
- **去重处理**：父子元素同时匹配时只保留子元素，避免重复高亮
- **毛玻璃浮层**：使用 `backdrop-filter` 实现半透明导航按钮，`z-index: 2147483647` 确保不被页面元素遮挡

## 兼容性

- Chrome / Chromium 系浏览器
- 支持普通静态页面及 React/Vue 等 SPA 框架渲染的页面
